🤝 API 사용 안내
이 앱은 GitHub Pages 정적 사이트예요. AI 기능을 쓰려면 당신 기기의 브라우저가 Anthropic 서버로 직접 요청을 보내야 해요.

그래서 API 키가 다음과 같이 처리돼요:
  • 서버(저)에게 전달되지 않음
  • 브라우저 sessionStorage 또는 localStorage에만 저장
  • 요청 시 HTTPS로 Anthropic에 전송

동의하면 "확인"을 누르세요. 동의하지 않으면 AI 기능은 비활성화돼요.
